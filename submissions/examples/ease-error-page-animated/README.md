# Animated Error Page (`ease-error-page-animated`)

A highly styled, modern 404/500 error page featuring a drifting background starscape, a floating vector astronaut character, glowing typography, and an interactive call-to-action button.

## Acceptance Criteria Met
- [x] **Large error code** with neon gradient glow (`ease-text-glow`).
- [x] **Animated character/illustration** - floating vector astronaut using keyframe-based translate/rotate animations.
- [x] **Error message and description** styled elegantly with custom typography.
- [x] **Back-home CTA** styled as an interactive button with scaling, radial hover glows, and shadow transitions (`ease-btn`).
- [x] **Subtle background animation** featuring shimmering/pulsing stars at variable delays.

## Files
- `demo.html`: Structure of the 404 space illustration page.
- `style.css`: Animation transitions, glowing filters, layout, and space theme colors.

## How to use
Incorporate the HTML structure into your standard 404 template. The astronaut character is fully self-contained as inline SVG and is animated entirely via pure CSS classes.
